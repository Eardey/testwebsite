Troubleshooting Common Issues

Standard steps for the technical issues we see most often — slow or crashing web apps, Wi-Fi that keeps dropping, and Lenovo laptops that have started to lag. Work through the relevant section below before logging a ticket with the help desk.

1. Web application issues

This applies mainly to Assurance, Tax, Strategy and Transactions, and Consulting staff working in browser-based tools. If a browser-based app is lagging or crashing, especially with large spreadsheets, clear your browser cache and confirm hardware acceleration is turned on.

2. Resetting your Wi-Fi connection

Persistent Wi-Fi drops can usually be fixed with a network reset from the Command Prompt.

To open it:

  1. Press the Windows key, or click Start.
  2. Type cmd (or "Command Prompt") into the search bar.
  3. Open the Command Prompt app.
  4. Run the following commands, in order:
ipconfig /release
ipconfig /renew
ipconfig /flushdns

Restart your computer once the commands finish so the changes take effect.

3. Windows Network Troubleshooter

If the manual reset above doesn't fix things, run the built-in Windows Network Troubleshooter instead.

Steps:

  1. Press Win + I, or open the Start menu → Settings.
  2. Go to Update & Security.
  3. Select Troubleshoot from the sidebar.
  4. Choose Additional troubleshooters.
  5. Click Internet Connections, then Run the troubleshooter.

If you're using (or plan to use) an Ethernet cable and run into issues there, skip the above and go straight to the help desk.

4. Lenovo laptop performance

For a slow or laggy Lenovo laptop, start with Lenovo Commercial Vantage to make sure your system, drivers, and firmware are current.

Steps:

  1. Open the Start menu and search for Lenovo Commercial Vantage.
  2. Go to Device → System Update.
  3. Click Check for updates.
  4. Install anything recommended, then restart if prompted.
